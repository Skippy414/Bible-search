# Installation and Setup Instructions for Developers

## Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed.
- Git should also be installed on your machine.

## Getting Started

1. **Clone the Repository**
   ```bash
   git clone https://github.com/<owner>/Bible-search.git
   cd Bible-search
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run the Application**
   ```bash
   npm start
   ```

## Running Tests

To run tests, use:
```bash
npm test
```
